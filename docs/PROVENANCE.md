# Provenance

The portable Feelings contract was extracted from the current Viventium implementation and QA on
July 18, 2026.

The band definitions, range semantics, decay formula, capsule behavior, typed reaction contract,
and dashboard information architecture were informed by the Feelings implementation inside the
MIT-licensed Viventium LibreChat fork. The standalone runtime is decoupled from LibreChat, Mongo,
JWT routes, React Query, Prompt Workbench, Telegram, voice, and Viventium's top-level FSL-licensed
orchestration/docs.

The upstream MIT notice is preserved in `THIRD_PARTY_NOTICES.md`. Top-level Viventium requirement
and QA prose is not copied wholesale into this Apache-2.0 repository.

Created by Adrien Beyk, creator of Project Viventium.

Research, implementation, testing, and editorial work used AI-assisted development under human
direction and review; attribution and responsibility remain with the project authors.
