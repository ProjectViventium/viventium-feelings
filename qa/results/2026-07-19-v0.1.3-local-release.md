## Honest capability boundary

This release supports Claude Code and Codex plugin hosts. It does not alter ordinary consumer
Claude or ChatGPT web conversations. Codex supplies plugin/composer identity but no arbitrary
third-party TUI status segment. Claude Code offers the owned `V Feelings` status line only after an
explicit user opt-in. The full Viventium macOS helper, not this plugin, owns the operating-system
status-menu surface.

Remote CI, the exact tagged archive, and fresh installs from the public GitHub URL must be recorded
after publication in a separate remote-release report.
